# Support

For ordinary users, start with:

- `START_HERE_RU.txt` / `START_HERE_EN.txt`
- `QUICK_START_RU.txt` / `QUICK_START_EN.txt`
- `TROUBLESHOOTING_RU.txt` / `TROUBLESHOOTING_EN.txt`

When reporting a problem, include:

- Which portable ZIP was used: CPU, GPU_FULL or GPU_LITE.
- Windows version and whether an NVIDIA GPU is present.
- The action that failed.
- Screenshots or exact error text if available.
- A support bundle if requested by the maintainer.

Support bundle command in portable builds:

```powershell
TunedImageSorter_CLI.exe --support-bundle --output <result_dir>
```

Diagnostics may contain folder paths, logs, reports and technical environment information. Review files before sharing them publicly.
