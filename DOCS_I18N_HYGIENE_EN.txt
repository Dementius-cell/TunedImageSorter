Tuned Image Sorter v69.6 / Stage 055 — first public polish / release identity pass

Purpose
-------
v69.6 is a first public polish / release identity pass on top of the confirmed rebrand base v69.3.1 / Stage 052.

This stage does not change application behavior. It is intended to clean up developer notes, bilingual RU/EN documentation and release-facing text before handing the project to another user or developer.

What is checked
---------------
PASS for this stage means:

1. Friend-ready user documents are present in RU/EN.
2. Developer-facing documents are present in RU/EN.
3. Stale references to earlier stages have been removed from developer notes and architecture notes.
4. RU documents do not contain large English prose blocks where Russian text is expected.
5. EN documents consistently use Stage 055 except for technical identifiers and compatibility strings.
6. VERSION.txt, README, RELEASE_FREEZE, RELEASE_GATE and RC_CHECKLIST are aligned with v69.6.
7. release-check includes the docs_i18n_hygiene_polish guard.

What is unchanged
-----------------
ML, face recognition, clustering, ordinary Start, apply-names, pipeline stages, SQLite schema, project.json, CSV schemas, Review clusters backend, result-health backend, support-bundle backend, CPU/GPU packaging split, pinned GPU runtime and bundled CUDA runtime are unchanged.
