Tuned Image Sorter v69.6 — where to begin
======================================

1. Extract the ZIP to a normal folder, for example:
   D:\TunedImageSorter_CPU
   D:\TunedImageSorter_GPU_FULL

2. For the shortest path, open:
   QUICK_START_EN.txt

3. For a non-technical first run, open:
   FIRST_RUN_EN.txt

4. Run the app:
   TunedImageSorter.exe

5. If Windows SmartScreen warns about an unknown app:
   More info -> Run anyway.
   This is normal for an unsigned portable exe.

6. In the GUI choose:
   input  = source photo folder;
   output = separate result folder outside input.

Important: do not place output inside input, or generated files may be scanned again later.

7. Before a large run:
   - run Environment check;
   - use Quick test;
   - first verify 20–50 photos.

8. Diagnostics use the console launcher:
   TunedImageSorter_CLI.exe --diagnostics-help
   TunedImageSorter_CLI.exe --runtime-preflight
   TunedImageSorter_CLI.exe --runtime-preflight --gpu
   TunedImageSorter_CLI.exe --release-check
   TunedImageSorter_CLI.exe --result-health --output <result_dir>
   TunedImageSorter_CLI.exe --support-bundle --output <result_dir>

9. If something goes wrong:
   ERRORS_EN.txt — what the error means and what to do.
   TROUBLESHOOTING_EN.txt — common portable package problems.

10. To send diagnostics to the developer:
   TunedImageSorter_CLI.exe --support-bundle --output <result_dir>
