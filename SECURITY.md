# Security Policy

## Supported Version

The current public handoff version is `v69.6`.

## Reporting

If you find a security or privacy issue, do not post private photos, support bundles or logs publicly. Open a private contact path with the maintainer if one is listed on the GitHub repository, or create a minimal public issue that does not include sensitive data.

## Privacy Model

Tuned Image Sorter is designed for local photo processing. The application workflow does not require uploading photos to a cloud service. Support bundles and diagnostics can still contain local paths, logs and report data, so share them carefully.
