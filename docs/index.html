Windows portable app - v69.6

Local photo sorting by detected face clusters

Tuned Image Sorter helps organize large folders of photos into face-based groups while keeping processing on the user's computer.

Приложение сортирует большие папки фотографий по распознанным лицам локально, без обязательного облачного сервиса.
