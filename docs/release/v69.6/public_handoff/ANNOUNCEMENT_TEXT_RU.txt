Короткий анонс
==============

Tuned Image Sorter v69.6 опубликован как первый public portable release для Windows.
Приложение сортирует большие папки с фотографиями по распознанным лицам и работает локально на компьютере пользователя.
Доступны три варианта: CPU, GPU_FULL для NVIDIA GPU и экспериментальный GPU_LITE.
Если не уверены, какую версию скачать, начните с CPU.
