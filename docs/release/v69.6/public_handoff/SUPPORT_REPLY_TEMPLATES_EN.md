# Support reply templates — EN

## Which version should I download?

If unsure, download `TunedImageSorter_CPU_portable_v69_6.zip`. It is the most compatible option and does not require an NVIDIA GPU.

## I have an NVIDIA GPU. What should I download?

Use `TunedImageSorter_GPU_FULL_portable_v69_6.zip`. It is large because it includes CUDA/cuDNN/cuBLAS runtime files.

## Is GPU_LITE the main version?

No. `GPU_LITE` is an experimental supplementary profile. The main options are `CPU` and `GPU_FULL`.

## The reports folder did not open after completion

This is a known non-blocking limitation in `v69.6`. Use the result/report open buttons in the UI.

## Do I need to send photos for diagnostics?

Usually no. Start with support-bundle / diagnostics. Before sharing a support bundle, remember that it may contain folder paths, logs and technical reports.
