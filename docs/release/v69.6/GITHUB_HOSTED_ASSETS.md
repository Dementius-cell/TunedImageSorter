# GitHub-Hosted Assets / Файлы, размещенные на GitHub

## English

The GitHub Release for Tuned Image Sorter v69.6 hosts:

- `TunedImageSorter_CPU_portable_v69_6.zip`
- `TunedImageSorter_GPU_LITE_portable_v69_6.zip`
- `SHA256SUMS.txt`

`TunedImageSorter_GPU_FULL_portable_v69_6.zip` remains an official profile, but it is larger than 2 GB and is intentionally not hosted in this GitHub Release. If the GPU_FULL build is distributed through another channel, use the matching line in `SHA256SUMS.txt` to verify it.
