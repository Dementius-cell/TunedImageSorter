# Tuned Image Sorter v69.6 Release Materials

This folder collects the public handoff material provided for the first public portable release.

## GitHub-Hosted Release Assets

- `TunedImageSorter_CPU_portable_v69_6.zip`
- `TunedImageSorter_GPU_LITE_portable_v69_6.zip`
- `SHA256SUMS.txt`

## Official Profile Not Hosted On GitHub

- `TunedImageSorter_GPU_FULL_portable_v69_6.zip`

GPU_FULL remains an official v69.6 profile, but the archive is larger than 2 GB and is intentionally not hosted in the GitHub Release. The checksum may still be kept in `SHA256SUMS.txt` for offline or alternative distribution verification.

## Included Handoff Text

The `public_handoff/` directory contains release draft text, publication checklists, announcement copy, support reply templates and SHA256 verification instructions in RU/EN.
